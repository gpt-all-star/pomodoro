# Pomodoro Timer Web Application

This is a simple web application implementing a Pomodoro Timer. The timer is programmable for a 25-minute work session as per the standard Pomodoro Technique. After each work session, a 5-minute break session automatically starts. After four consecutive work sessions, a 15-minute break session automatically starts. The application has sound notifications at the transition between work and break sessions, and also displays the number of completed work sessions.

## Technology Stack
- HTML
- CSS3
- JavaScript
- Node.js
- SQLite

## Requirements
- Node.js
- npm

## How to Run the Project
1. Install the dependencies by running `npm install`.
2. Start the server by running `node server.js`.
3. Open a web browser and go to `http://localhost:3000`.
